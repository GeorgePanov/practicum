# Машинное обучение в бизнесе.
## Описание проекта
Вы работаете в нефтедобывающей компании и вам поручили решить, где бурить новую скважину. 
Шаги для выбора локации:
- В выбранном регионе собирают характеристики для скважин: качество нефти и объём её запасов;
- Строят модель для предсказания объёма запасов в новых скважинах;
- Выбирают скважины с самыми высокими оценками значений;
- Определяют регион с максимальной суммарной прибылью отобранных скважин.

Даны пробы нефти в трех регионах. Характеристика уже известна. 

## Цель
Построить модель для выбора наиболее прибыльного региона. Также нужно проанализировать возможную прибыль и риски техникой Bootstrap.
